
# House Price Prediction using Linear Regression

This project implements a **Linear Regression model** to predict house prices based on key features such as square footage, number of bedrooms, and number of bathrooms. The model is built using Python and the Kaggle House Prices dataset.

## Dataset
The dataset used for this project is from the Kaggle competition:
House Prices – Advanced Regression Techniques.

Important features used in this project:
- GrLivArea (Above ground living area in square feet)
- BedroomAbvGr (Number of bedrooms)
- FullBath (Number of bathrooms)
- SalePrice (Target variable)

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Steps Performed
1. Imported required Python libraries.
2. Loaded the Kaggle housing dataset.
3. Selected important features for prediction.
4. Split the dataset into training and testing sets.
5. Implemented Linear Regression model.
6. Predicted house prices using the trained model.
7. Evaluated the model using Mean Absolute Error and R² score.
8. Visualized the results using a scatter plot.

## Results
The model predicts house prices based on living area, bedrooms, and bathrooms. The performance was evaluated using regression metrics such as MAE and R².
